# AI Lab 9: Fruit Classification and Analysis

This repository contains my AI Lab 9 work on fruit dataset analysis and classification using Python in Jupyter Notebook.

## Lab Objective

The goal of this lab is to:
- explore the fruit dataset,
- understand feature behavior,
- prepare data for machine learning,
- and perform fruit label prediction based on physical measurements.

## Dataset

Source file:
- fruit_data_with_colors .csv

Main columns used:
- fruit_label
- fruit_name
- fruit_subtype
- mass
- width
- height
- color_score

Note:
- The dataset filename includes a space before .csv. Use the exact same name when loading it.

## Repository Contents

- fruit_data_with_colors .csv: dataset used in this lab
- mehrali_24p0500_lab9.ipynb: notebook containing all analysis and model steps
- mehrali_24p0500_lab9.html: HTML export of the notebook
- mehrali_24p0500_lab9.pdf: PDF export of the notebook
- lab9.pdf: lab handout/instructions

## Workflow Summary

1. Load and inspect the dataset.
2. Handle missing or unusual values where needed.
3. Explore data through descriptive statistics and visual analysis.
4. Prepare feature matrix and target labels.
5. Train and evaluate a classification model.
6. Interpret outputs and document findings.
